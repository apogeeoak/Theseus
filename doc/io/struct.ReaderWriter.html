pub struct ReaderWriter<IO> { /* fields omitted */ }
Expand description

A readable and writable “stateful” I/O stream that keeps track of its current offset within its internal stateless I/O stream.

This implements the [bare_io::Read] and [bare_io::Write] traits for read and write access, as well as the [bare_io::Seek] trait if the underlying I/O stream implements KnownLength. It also forwards all other I/O-related traits implemented by the underlying I/O stream.
